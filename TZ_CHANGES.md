# Доработки по ТЗ

В проект добавлены модули проверки ИНН, прогноза погоды и тепловой карты склада без удаления существующих сценариев.

## Что изменено

- Форма отгрузки:
  - добавлен ввод ИНН контрагента и кнопка проверки по API;
  - добавлено отображение статуса, организации и даты проверки;
  - при изменении ИНН результат проверки сбрасывается;
  - оформление отгрузки блокируется без успешной проверки или при статусе «В черном списке»;
  - добавлен блок города/даты доставки и получение прогноза Open-Meteo;
  - результат проверки погоды сохраняется и связывается с отгрузкой после сохранения.

- Форма поставки:
  - добавлен ввод ИНН поставщика и кнопка проверки;
  - поставка блокируется без успешной проверки ИНН;
  - результат проверки сохраняется и связывается с поставкой.

- Главное меню:
  - для администратора добавлен пункт и плитка «Карта склада»;
  - для кладовщика карта скрыта, прямой доступ блокируется сообщением «Нет доступа к карте склада».

- Карта склада:
  - реализована сетка A1–H6;
  - цвета рассчитываются по сроку годности и количеству с приоритетами из ТЗ;
  - реализована синяя рамка для низкого количества при сроке от 15 дней;
  - добавлены легенда, сводка, поиск товара, подсветка найденных ячеек и информация по клику;
  - данные перечитываются кнопкой «Обновить» без закрытия формы.

- База данных:
  - добавлены таблицы `contractor_checks`, `blacklisted_inns`, `blocked_operation_logs`, `weather_checks`;
  - в `product_batches` добавлена колонка `cell_code`;
  - приложение выполняет мягкую инициализацию схемы при запуске, также приложен `database_tz_patch.sql`.

## Настройка внешней проверки ИНН

Сервис проверки использует DaData, если задана переменная окружения `DADATA_TOKEN` или `DADATA_API_KEY`.
Для учебного запуска без ключа включен демонстрационный режим: валидный ИНН получает внешний статус `LOCAL_DEMO_ACTIVE`, а локальный черный список `blacklisted_inns` продолжает работать.

## Важное

Проверить сборку в этой среде не удалось, потому что здесь не установлен .NET SDK (`dotnet` отсутствует). Код подготовлен под исходный `net8.0-windows` WinForms-проект.
